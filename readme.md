## Installation

1.  Clone *Orbiter* to your server:

		git clone git://github.com/kasparsd/orbiter.git orbiter
		
2. 	Copy `config_sample.ini` to `config.ini`:

		cp config_sample.ini config.ini
		
	and adjust the settings to fit your environment.
	
3. 	Visit your site.
